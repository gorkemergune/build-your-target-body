# Build Your Target Body

## Project Vision

Build Your Target Body is an AI-powered body transformation platform designed to help users achieve their desired physique through structured tracking, intelligent insights, and personalized coaching.

The application supports:

* Weight loss
* Weight gain
* Body recomposition
* Muscle gain

The system tracks:

* Weight
* Body fat percentage
* Body measurements
* Calories consumed
* Calories burned
* Daily nutrition logs
* Workout history
* Progress photos
* User notes

The platform provides visual analytics, prediction models, and AI coaching powered by Gemini.

## Architecture Philosophy

The AI coach must never replace actual tracking.

Tracking features should work independently from Gemini.

AI is an enhancement layer.

The application must remain useful even if Gemini is disabled.
